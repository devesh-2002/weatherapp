# weatherapp
This is a simple site shows Weather of the place which user inputs. 
## Tech Stacks
1. HTML
2. CSS
3. React.js
## Run the project
1. Fork the project.
2. Clone the forked repository.
```git clone 
https://github.com/<username>/weatherapp 
```
3. Open git bash.
4. Run the project on ```http://localhost:3000/```
